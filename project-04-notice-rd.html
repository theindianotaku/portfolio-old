Description

After the launch of the first version of the Channel I Noticeboard for android, there was a need to increase the utility of the app and upgrade it from a MVP to a full-featured version with respect to its web version.

The app was not user centric previously. So our first aim was to look for utility that a user would expect from a content heavy app like this. The ability to sort notices based on filters, the ability to save important notices and also set a reminder for event/deadline specific notices were the three features that were given the utmost importance in implementation in this revision. We planned out several other features to manage push notifications in a customizable format.

There were improvements in the background behaviour of the app like improvements in caching and offline storage of the notices, an infinite scroll and fix several load crashes.

The app had a similar navigation drawer based categorization and a bottom tab switch which separated the current and expired notices in the selected category. The cards had a slight weight plus contrast change to make the differentiation of read and unread messages easier. The web view employed in viewing the PDFs was improved and a zoom feature was added.

The app was successfully launched in the assigned period of time. Currently after several user reviews, we are planning another revision to the app which would include various quality improvements to the code.
