Description

Qiosk empowers you to seamlessly discover, save and share professional articles. Create collections that move your industry forward. My role was the front-end development of the Web App and their Admin App with Angular JS as the structural framework. The back-end stack of this project included Django REST framework.

First step of the development process was creating the HTML templates using the mockups and the stylesheet guidelines that they have provided. The design being non-native to any of the currently available design languages of the CSS frameworks, I structured the stylesheets for the UI elements and used the Bootstrap v3 Grid for the layouting. The design included a dynamic cards view which had to be vertically laid out. Masonry suited the purpose perfectly. Still new to the concept of managing asynchronous tasks in initializing UI elements, I found it difficult to initiate masonry just after the page got populated with HTML elements.

UI-Router was used instead of the generic ngRoute as it allowed multiple/nested views. The basic ‘$state’ notion is a very powerful concept when compared to assigning controllers and templates to URL routes as in the case in ngRoute.

No project is without any flaws or areas which need further development. That time, with having yet to understand the concept of ‘custom-directives’ I did almost neglect it. Now looking back at the state of the project, using directives should have been my prime route for handling the DOM elements.
